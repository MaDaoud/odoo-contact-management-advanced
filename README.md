# Odoo Contact Management Advanced

This project is an advanced Odoo module for managing contacts.  
It was developed as part of academic labs (TP2, TP3, TP4).

## Features
- Contact CRUD management
- Contact categorization (Many2one)
- Relationships between contacts (Many2many)
- PDF report generation
- Access control rules (security)
- REST API (GET / POST)
- Dynamic filters and search
- Kanban dashboard for categories

## Requirements
- Odoo 19
- Python 3.10+
- PostgreSQL

## Installation
1. Clone the repository into your Odoo `addons` directory
2. Restart the Odoo server
3. Activate developer mode
4. Install the module from Apps

## REST API
- GET: `/mon_module/contacts`
- POST: `/mon_module/contacts`
